Diablo IITC Plugins

Copy any URL below and paste it into IITC Button as a custom plugin.

How to add a plugin in IITC Button (one-time per device):
  1. Click the IITC Button icon → Settings (gear icon)
  2. Go to the Plugins tab → Add plugin by URL (or User plugins)
  3. Paste the raw GitHub URL from below and confirm
IITC Button fetches the script from GitHub on each load — so after you run ./sync-plugins.sh on your Mac, every device picks up the change automatically.

Stop using local file paths — replace any file:// entries with the GitHub URL below for the same plugin.
